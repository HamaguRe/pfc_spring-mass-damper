# PFCによるバネ・マス・ダンパ系の位置制御

PFC（Predictive Functional Control）により1自由度バネ・マス・ダンパ系の
位置制御を行います。

オープンループで設計しているため入力制約を設けてもそれなりに上手く制御できます。

## 実行方法

```
$ cargo run && python3 data_plot.py
```
